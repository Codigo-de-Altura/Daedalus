## How to use

Daedalus is keyboard-driven and consistent: the same key always does the same thing, everywhere. If you forget a shortcut, open the contextual help from any screen to see exactly which keys are available right now.

- **Consistent shortcuts** — navigation and common actions (move, enter, back, root, quit) keep the same keys across every area.
- **Contextual help** — a help bar shows the most relevant shortcuts at all times, and an expanded help view lists everything available in the current context (area, screen, or form).
