# `daedalus init` — Usage Guide

> Audience: end users of Daedalus. Authored and maintained by **C-3PO** as the feature is implemented and validated. This is the initial outline to be filled with real behavior once the ticket passes validation.

## Overview

_To be completed by C-3PO after implementation._

`daedalus init` creates the canonical `.daedalus/` workspace inside your target repository — the single source of truth for your project's AI structure (agents, prompts, workflows, SDD backlog).

## How to use

_Steps the end user follows._

1. Open a terminal at the root of your target repository.
2. Run `daedalus init`.
3. Daedalus creates the `.daedalus/` workspace with its standard folder structure.
